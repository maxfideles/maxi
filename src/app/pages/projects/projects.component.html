Sunfácil

Personal Project

Sales management web system for a medium-sized travel agency in my hometown.

Problem:

With two offices in different cities and numerous travel agents, the owner responsible for sales management used to receive an Excel spreadsheet from each employee at the end of the month to process profits and commissions. This system caused delays and inconsistencies, as employees could input and edit information freely, leading to a lack of standardization and reliability.

Solution:

A web-based system that centralizes sales records in a database, with role-based authentication and authorization for travel agents and management staffs.

More info

  • An end-to-end application was developed using full-stack technologies, including TypeScript/Angular for the frontend, Java/Spring Boot for the backend, and Swift for the native iOS app, enabling a consistent experience across multiple platforms;
  • The application features a highly accurate simulator for analyzing photovoltaic systems in over 5,000 Brazilian cities, employing advanced engineering methods to compete effectively with global market leaders;
  • A vast amount of solar irradiation data from all Brazilian cities, collected by the National Institute for Space Research, was processed using Business Intelligence (BI) tools and optimally stored in a dedicated PostgreSQL database;
  • Effective communication between the system's components was established via RESTful API requests, ensuring smooth and efficient integration between the frontend and backend. This approach not only ensures interoperability between the different project elements but also promotes robust scalability and simplified maintenance throughout the application's lifecycle;
  • Swagger was used to automate the generation of API documentation, ensuring accurate and up-to-date documentation;
  • Deployment was carried out in an efficient and organized manner, using the Railway platform for the backend and Netlify for the frontend, ensuring fast and stable project delivery.

WJR Turismo

Freelancing

Sales management web system for a medium-sized travel agency in my hometown.

Problem:

With two offices in different cities and numerous travel agents, the owner responsible for sales management used to receive an Excel spreadsheet from each employee at the end of the month to process profits and commissions. This system caused delays and inconsistencies, as employees could input and edit information freely, leading to a lack of standardization and reliability.

Solution:

A web-based system that centralizes sales records in a database, with role-based authentication and authorization for travel agents and management staffs.

More info

  • A comprehensive full-stack application was developed using cutting-edge technologies. The frontend was built with TypeScript and Angular, while the backend utilized Java with Spring Boot, providing users with a robust and integrated experience;
  • A highly secure authentication and authorization system based on roles was implemented, integrating Spring Security with Auth0 JWT tokens to secure the API. This ensures that only authorized users have access to specific resources;
  • On the frontend, precise access control based on user roles was established using Angular's Auth Guard, ensuring that only users with the appropriate permissions can access certain parts of the application. This guarantees security and data integrity;
  • Additionally, the user experience was optimized by implementing Progressive Web Application (PWA) functionality, allowing users to access the application quickly and efficiently, even under adverse network conditions, and offering a native experience on mobile devices;
  • The deployment was carried out in an efficient and organized manner, using the Railway platform for the backend and Hostinger for the frontend, ensuring a fast and stable delivery of the project.
